# Pratikum-2

Luas Lingkaran

Diketahui = a. phi= 3,14
	    b. r = 10

Ditanya = Luas Lingkaran ?
Jawab = phi*r*r
      = 3,14*10*10
      = 314

Keliling Linkarang

Diketahui = a. phi= 3,14
	  = b. r = 20

Ditanya = Keliling Lingkaran ?
Jawab = 2*phi*r
      = 2*3,14*20
      = 125,6
